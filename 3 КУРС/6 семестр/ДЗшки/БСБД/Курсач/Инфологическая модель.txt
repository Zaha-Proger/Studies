Сущности:
1. Пациенты
2. Врачи
3. Медицинские осмотры
4. Результаты медицинских осмотров

Атрибуты:
1. Пациенты:
- ID пациента
- ФИО
- Пол
- Адрес
- Номер телефона
- Электронная почта

2. Врачи:
- ID врача
- ФИО
- Специализация
- Номер телефона
- Электронная почта

3. Медицинские осмотры:
- ID осмотра
- ID пациента
- ID врача
- Дата и время осмотра
- Описание состояния пациента
- Рекомендации врача

4. Результаты медицинских осмотров:
- ID результата
- ID осмотра
- Дата и время получения результата
- Описание результата

Связи:
1. У каждого пациента может быть несколько медицинских осмотров, но каждый осмотр относится только к одному пациенту.
2. У каждого врача может быть несколько медицинских осмотров, но каждый осмотр проводится только одним врачом.
3. У каждого медицинского осмотра может быть несколько результатов, но каждый результат относится только к одному осмотру.
