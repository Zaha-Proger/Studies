What's New

Users of previous versions of Packet Tracer will note a variety of new features in this version of Packet Tracer.

Workspace


  • New themes
  • Aligned workspace objects
  • Ability to deploy devices to a specific rack
  • Accessibility update
  • Ability to name a rack

Other Changes


  • IP OSPF Point-to-Point on Ethernet interface
  • IPv6 OSPF Point-to-Point on Ethernet interface
  • Upgraded PTSA and PTMO support
  • Various enhancements to existing protocols

New Devices


  • Cisco 4331 Integrated Services Router
  • Cisco Wireless Lan Controller 3504
