WebEngine Views Accessibility


The following table shows the compatibility matrix between the webview UI in Cisco Packet Tracer and the default screen text narration clients on different platforms. A limited set of keyboard keys we have tested with is also provided.

End users are encouraged to explore other platform-specific keyboard shortcuts. We are also interested in end-user feedback regarding their experience with these features. This information may help our team to make Cisco Packet Tracer more accessible in its future releases.
