INTEGER - целочисленный, 32-битное чисдл
SMALLINT - 16-битное целое
NUMERIC - дробное число, 2 цифры после запятой
DECIMAL -
REAL - вещественные числа, состоящие из десятичной точки и экспоненты
DOUBLE PRECISION - компьютерный формат представления числа с плавающей запятой, занимающий в памяти 64 бита
FLOAT - числовые значения, которые могут быть нецелыми
CHAR - тип данных, предназначенный для хранения одного символа (управляющего или печатного) в определённой кодировке
VARCHAR - представляет строку переменной длины
CLOB - предназначен для хранения больших текстовых данных заданной кодировки
NCHAR -  Используется для хранения строк переменной длины, состоящих из символов в кодировке Unicode.
NCHAR VARYING -
NCLOB - предназначен для хранения больших текстовых данных в кодировке unicode
BIT-  используется для хранения булевого (логического) значения, 0 или 1
BIT VARYING - могут иметь переменную длину, но не превышающую n ; строки большей длины не будут приняты. Запись bit без указания длины равнозначна записи bit(1) , тогда как bit varying без указания длины подразумевает строку неограниченной длины.
BLOB - массив двоичных данных. В СУБД BLOB — специальный тип данных, предназначенный, в первую очередь, для хранения изображений, а также компилированного программного кода.
DATE - используется для величин с информацией только о дате в формате 'YYYY-MM-DD 
TIME - тип данных для хранения времени (3 байта)
TIME STAMP -  тип данных для хранения даты и времени. Данные хранятся в виде количества секунд, прошедших с начала «эпохи Юникса» (4 байта)
TIME WITH TIME ZONE - тип данных для хранения времени с часовым поясом
TIMESTAMP WITH TIME ZONE - тип данных для хранения даты и времени с часовым поясом
INTERVAL DATE - используются для хранения разницы между двумя значениями даты-времени.
INTERVAL TIME - используются для хранения разницы между двумя значениями времени.
BOOLEAN - примитивный тип данных в информатике, которые могут принимать два возможных значения, иногда называемых истиной ( true ) и ложью ( false )
ARRAY - тип данных для хранения массива значений, идентифицируемых по индексу строки и столбца
